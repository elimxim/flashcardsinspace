### Prerequisites
- Java 24+
- Node.js 22+
- PostgreSQL

### Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/elimxim/flashcardsinspace.git
   cd flashcardsinspace
   ```

2. Configure the database in `props/application.yaml`

3. Build and run:
   ```bash
   ./gradlew bootRun
   ```

4. For frontend development:
   ```bash
   cd src/main/vue
   npm install
   npm run dev
   ```
