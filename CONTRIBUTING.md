## Sign-off

Add a sign-off line to each commit to certify you accept the terms above:

```
Signed-off-by: Your Name <your.email@example.com>
```

You can do this automatically with `git commit -s`.

## How to contribute

1. Open an issue first to discuss substantial changes.
2. Fork the repository and create a branch for your change.
3. Follow the existing code style (see [`.claude/CLAUDE.md`](.claude/CLAUDE.md) for
   project conventions).
4. Run the test suite (`./gradlew test`) before submitting.
5. Open a pull request with a clear description, including the sign-off line.
